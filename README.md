# Ticket PDF Generator

ブラウザだけでチケット入稿用のPDF下地を作る静的Webアプリです。  
HTML Canvasでプレビューし、`pdf-lib` でPDFを書き出します。

## できること

- チケットサイズのプリセット選択 / カスタム指定
- 塗り足し 3mm 固定
- トンボ表示
- ミシン目位置の指定
- 本券 / 半券レイアウトのプレビュー
- 連番のプレースホルダー入稿
- アプリ内での連番付きPDF生成
- 設定の自動保存
- 設定JSONの書き出し / 読み込み

## 使い方

1. `index.html` をブラウザで開きます。
2. 左側のアコーディオンでサイズ、ミシン目、文字、連番を調整します。
3. 右側のプレビューで見た目を確認します。
4. `PDFを生成` を押して `ticket.pdf` を出力します。

## 連番モード

連番設定には2つのモードがあります。

### 1. 印刷所で差し込み

- `###` をプレースホルダーとしてPDFに残します
- PDFは1ページのみ生成します
- 印刷所向けコメントをそのまま依頼文として使えます

### 2. このアプリで連番PDFを作る

- `001` や `015` のような開始番号を指定します
- 発行枚数ぶんのPDFページを生成します
- プレビューには1枚目の番号を表示します

## 設定保存

- フォームの内容は `localStorage` に自動保存されます
- 保存データには `schemaVersion` を持たせています
- 古い保存データは読み込み時にマイグレーションされます

## 設定ファイル

- `設定を書き出し` で `ticket-settings.json` を保存できます
- `設定を読み込み` で別ブラウザや別端末でも同じ設定を再利用できます

## 技術メモ

- フレームワークなし
- HTML / CSS / Vanilla JavaScript
- PDF生成ライブラリは `pdf-lib` を使用
- UIフォントは Google Fonts から読み込み
- PDF内テキストはCanvas描画結果を画像として埋め込む方式です

## 注意

- このツールは入稿補助用です
- 最終入稿前にサイズ、塗り足し、ミシン目位置、連番指示を必ず確認してください
- 印刷所ごとにトンボや加工指示の運用が異なる場合があります
